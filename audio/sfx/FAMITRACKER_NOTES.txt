MITOSIS PANIC - FamiTracker SFX Creation Notes

FILE: game_sfx.ftm (to be created)

This file documents how to create all sound effects in FamiTracker.
Actual .ftm file creation requires FamiTracker software installed.

==============================================================================
PROJECT SETUP
==============================================================================

1. Launch FamiTracker
2. File → New
3. Set parameters:
   - Machine: NTSC
   - Expansion chip: None (2A03 only)

4. Module → Module Properties:
   - Speed: 6 (default NTSC, 60 Hz)
   - Tempo: 150 BPM (for timing calculations)

==============================================================================
STRUCTURE
==============================================================================

Create 8 separate patterns/songs, one for each sound effect:

Pattern/Song 0: SFX_MITOSIS (Cell division)
Pattern/Song 1: SFX_NUTRIENT_A (Green nutrient)
Pattern/Song 2: SFX_NUTRIENT_B (Yellow nutrient)
Pattern/Song 3: SFX_NUTRIENT_C (Pink nutrient)
Pattern/Song 4: SFX_ANTIBODY_WARN (Warning)
Pattern/Song 5: SFX_DAMAGE (Damage pulse)
Pattern/Song 6: SFX_DAMAGE_NOISE (Damage noise)
Pattern/Song 7: SFX_GAME_OVER (Game over jingle)

==============================================================================
INSTRUMENT DEFINITIONS
==============================================================================

Instrument 00 - SFX Pulse Soft (25% duty)
---------------------------------------
Type: 2A03 Pulse
Volume envelope: F F E D C 8 4 2 0
Arpeggio: None
Pitch: Use Qxy/Rxy effects for sweeps
Duty: 0 (25%)

Instrument 01 - SFX Pulse Standard (50% duty)
-------------------------------------------
Type: 2A03 Pulse
Volume envelope: F F E E D C 8 4 2 0
Arpeggio: None
Pitch: Use Qxy/Rxy effects for sweeps
Duty: 1 (50%)

Instrument 02 - SFX Pulse Harsh (75% duty)
----------------------------------------
Type: 2A03 Pulse
Volume envelope: F F E E D D C 8 4 2 0
Arpeggio: None
Pitch: Use Qxy/Rxy effects for sweeps
Duty: 2 (75%)

Instrument 10 - SFX Triangle
-------------------------
Type: 2A03 Triangle
(Use note length for envelope control)

Instrument 20 - SFX Noise Short
----------------------------
Type: 2A03 Noise
Volume envelope: F F E D C A 8 4 2 0 (quick decay)
Pitch: Variable (use period 1-10)

==============================================================================
EXPORT PROCESS
==============================================================================

After all SFX are complete:

1. File → Export as NSF
   - Name: game_sfx.nsf
   - Include: All songs (0-7)
   - Save to: audio/sfx/

2. Convert to FamiTone2 format:
   > text2data game_sfx.nsf -sfx

   (Note: -sfx flag tells converter these are sound effects)

3. Move to exports:
   > copy game_sfx.s ../exports/sfx_data.s

4. Verify:
   - Check all 8 effects are present in data
   - File size should be 500 bytes - 2 KB
   - Each effect has proper channel assignments

==============================================================================
TESTING
==============================================================================

In FamiTracker:
- Play each SFX individually
- Verify timing (all under 0.3s except game over)
- Check volume levels are appropriate
- Ensure no harsh/unpleasant tones
- Test nutrient variants sound distinct

In-game (after integration):
- Play during gameplay to verify they cut through music
- Test multiple SFX playing simultaneously
- Verify damage effects (SFX 5+6) play together correctly
- Check game over jingle plays completely
